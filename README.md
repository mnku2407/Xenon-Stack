## Setup

open terminal and navigate to assignment and run

```bash
  npm i 
```

After that you have to run another command for start localhost server

```bash
  node index.js
```



## Tech Stack

**Client:** HTML, CSS, JAVASCRIPT

**Server:** Node, Express, mongodb etc
